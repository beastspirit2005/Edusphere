# EduSphere 🎓🤖

EduSphere is an **AI-powered desktop learning platform** built with **Electron**.
It provides intelligent tutoring and AI assistance for both **students and teachers**, creating a smarter and more interactive learning environment.

## ✨ Features

* 🤖 **AI Tutor for Students**
* 🧑‍🏫 **AI Assistant for Teachers**
* 💬 **Integrated AI Chat System**
* ⚡ **Fast Electron-based Desktop Application**
* 🧠 **Local AI Integration using Ollama (Llama models)**
* 📚 Interactive learning tools

## 🛠️ Tech Stack

* **Electron.js**
* **HTML / CSS / JavaScript**
* **Ollama**
* **Llama 3 AI Model**

## 🚀 Installation

Clone the repository:

git clone https://github.com/beastspirit2005/EduSphere.git

Navigate to the project folder:

cd EduSphere

Install dependencies:

npm install

Run the application:

npm start

## 🎯 Vision

EduSphere aims to bring **AI-powered education directly to the desktop**, helping students learn faster and teachers teach smarter.
