# Edusphere
An AI-Powered Student Management & Learning Platform

EduSphere is a full-stack student platform designed to simplify academic management and enhance learning using an integrated AI assistant. It provides a secure environment where students and teachers can interact, manage academic records, upload documents, and get instant AI-based assistance.
